如何做一个个人博客1.0

2025/1/12

1.进入 https://hexo.io/zh-cn/ 。

2.安装git，node.官网下载慢进https://cwaz.lanzoul.com/i3f7s2koy0ab。

3.新建文件夹：hexo

4.右键，点open git bash here.

5.输入npm install hexo-cli -g hexo init blog cd blog npm install hexo s

6.进入localhost:4000，如果你可以进入。那么你已经有一个个人博客了！但别人不能看到。

7.输入：hexo new “文章内容自己起”

8.进入hexo\blog\source_posts，你会看到”hello,world.md”和”你刚才起的文章.md”。用记事本打开”你刚才起的文章.md”。

9.输入你想写的文章，前面一定要写###，换行，输入“”。

10.保存。在git写 hexo g。

11.在 https://hexo.io/zh-cn/ 下滑进入416主题.

12.选择一个主题。复制install/安装下面的内容，并在git上输入。用记事本进入_config.yml，找到theme:，写出这个主题的名称，在终端hexo g。

13.注册GitHub账号,点击右上角的+，进入new repository。输入“账号名+github.io”，并加readme.md。在git输入git config –global user.name “账号名”,ssh-keygen -t rsa -C “邮箱”（询问全回车），cat ~/.ssh/id_rsa.pub(复制结果）。进入头像-设置-ssh-new ssh-粘贴ssh.

14.在_config.yml中type:””中加上git，最后加上repository:(空格）（进入code，SSH下面复制。格式：git@github.com:xxx/xxx.github.io.git）（在type下面加）

15.git输入：npm install hexo-deployer-git –save 最后部署博客：hexo g && hexo d. 如果没有报错，你可以在搜索栏上输入“账号名+github.io”，可能连接不上，多加载10-20次，如果可以，那么就大功告成了！
